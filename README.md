# Projeto Quiz
Um pequeno projeto de quiz desenvolvido para praticar HTML.

## 📚 Sobre o projeto
Este projeto faz parte dos meus estudos de desenvolvimento web.
A ideia é criar um quiz simples com perguntas e respostas utilizando HTML.

## 🚀 Tecnologias utilizadas
- HTML5 (por enquanto)

## 🎯 Objetivo
Praticar conceitos básicos de HTML como:
- formulários
- inputs
- estrutura de página

## 📂 Como executar
1. Baixe ou clone este repositório
2. Abra o arquivo 'index.html' no navegador
